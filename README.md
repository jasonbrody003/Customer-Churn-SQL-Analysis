# Customer Churn Analysis using SQL

## Project Overview
This project analyzes a telecom customer dataset to identify patterns influencing customer churn.

## Dataset
Telco Customer Churn Dataset containing 7043 customers.

## Key Analysis
- Customer churn distribution
- Churn by contract type
- Monthly charges comparison
- Customer tenure analysis

## Tools Used
- SQL
- SQLite Online

## Key Insights
- 26.5% of customers churned
- Month-to-month contracts show the highest churn
- Customers who churn pay higher monthly charges
- New customers churn more frequently
